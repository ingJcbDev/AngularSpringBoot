# Angular y Spring Boot

Version node

```bash
node -v
```

Version npm

```bash
npm -v
```

Install typescript para tener la sintaxis

```bash
npm install -g typescript
```

Version ts

```bash
tsc -v
npm list typescript -g
```

Install Angular

```bash
npm install -g @angular/cli
```

Version Angular
```bash
ng version
```

Crear un nuevo proyecto en angular

```bash
ng new clientes-app
```

## server angular

encender y abriri en el navegador

```bash
ng serve -o
```
